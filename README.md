# اپلیکیشن شیوه‌نامه‌های جشنواره

این پروژه یک برنامه آفلاین اندروید است که ۱۲ شیوه‌نامه PDF، نمایشگر داخلی PDF و جست‌وجوی یکپارچه فارسی را در خود دارد.

## امکانات نسخه ۱.۱

- صفحه معرفی رسمی واحد فناوری‌های دیجیتال قطب کشوری نجوم و فناوری‌های فضایی
- راهنمای انتخاب مسابقه بر اساس مقطع تحصیلی و حوزه موردعلاقه
- فهرست ۷۷ گرایش و پیوند مستقیم به صفحه آغاز شرایط هر گرایش
- بزرگ‌نمایی دو انگشتی، دکمه‌های بزرگ‌نمایی و تطبیق با عرض صفحه
- جست‌وجوی بهبودیافته فارسی با یکسان‌سازی «ی/ي»، «ک/ك»، نیم‌فاصله و اعراب
- حرکت بین نتایج قبلی و بعدی جست‌وجو
- رفتن مستقیم به شماره صفحه
- ذخیره اصل PDF در گوشی
- اشتراک‌گذاری اصل PDF
- چاپ یا ذخیره خروجی از طریق سامانه چاپ اندروید
- حفظ اصلاحیه فونت PDF با `disableFontFace` و `useSystemFonts`

اطلاعات راهنمای انتخاب در فایل زیر نگهداری می‌شود:

```text
app/src/main/assets/eligibility-index.json
```

مبنای اطلاعات، جدول مقاطع و بخش «شرایط شرکت‌کنندگان» در هر یک از ۱۲ شیوه‌نامه است. متن کامل شیوه‌نامه، مرجع نهایی شرایط اختصاصی هر گرایش خواهد بود.

## ساخت APK با GitHub

1. در GitHub یک مخزن جدید بسازید.
2. تمام محتویات این پوشه را در ریشه مخزن بارگذاری کنید.
3. وارد زبانه **Actions** شوید و اجرای Workflow را تأیید کنید.
4. فرایند **Build Android APK** را باز و گزینه **Run workflow** را بزنید.
5. پس از پایان اجرا، در بخش **Artifacts** فایل `festival-guidelines-apk` را دریافت کنید.
6. فایل ZIP را باز کنید؛ فایل `app-debug.apk` آماده نصب آزمایشی است.

برای انتشار عمومی در فروشگاه‌ها باید نسخه release امضاشده ساخته شود؛ نسخه debug برای نصب مستقیم و آزمایش مناسب است.
